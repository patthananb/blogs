About This Blog

This blog is built with simplicity in mind. All posts are written in Markdown, making it easy to focus on writing without worrying about formatting.

How to Add New Posts

  1. Create a new .md file in the posts/ folder
  2. Write your content in Markdown
  3. Add the post info to js/posts.js
  4. That's it! Your post will appear on the homepage
